## What is enex
enex is a game in wich you are given a start wikipedia page and need to find a way to get to another given wikipedia page only using the links on the page itself with the least possible steps

Because this game was made as a school project it only features the german language for both the game and wikipedia - feel free to fork and translate it

<!-- TOC --><a name="how-to-playgamerules"></a>
## how to play/gamerules
depending on the game rules you selected prior to playing, you either start with a random or a specific wikipedia page and need to find the shortest path to the either random or chosen target page. You do this by clicking links on pages until you find the target page or if enabled the time is up. if you ever want to go back, ENEX inclueds a path tree of pages you have discoverd and the distance from the start to said page. Once you found a path enex will give you a score based on the amount of steps your (shortest) solution has and the time you took to find it. it will also show *one* ยน ideal solution as an example


ยน: because of how closely wikipedia is linked it is possible and not uncommon for multiple optimal solutions to exist. to save time and recouces enex will only show the one it founds first; you can use pages like [six degrees of wikipedia](https://www.sixdegreesofwikipedia.com) to view different solutions (just don't use it to cheat ;3)
